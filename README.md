## ✨ Features

- **Natural Conversations** — Talk to TSM-BOT like a friend
- **Task Management** — Organize your day with to-dos and reminders
- **Smart Research** — Get concise answers and summaries instantly
- **Content Creation** — Draft emails, notes, and brainstorm ideas
- **Private & Secure** — End-to-end encryption for all your data
- **Works Everywhere** — Seamless sync across all devices

## 🛠️ Getting Started

```bash
# Clone the repo
git clone https://github.com/Zeus-Vibes/TSM-BOT.git
cd TSM-BOT

# Install dependencies
npm install

# Start dev server
npm run dev
```

Open **http://localhost:5173** in your browser.
